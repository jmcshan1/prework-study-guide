Prework Study Guide

HTML

  • The head element contains information about the webpage.The user cannot see what is contained in the head element.
  • The body element represents the visible content shown to the user.
  • The ul,ol, and li represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displated on the webpage.
  • The header element appears at the top of the page and usually contains a navigation menu and/or introductory content.
  • The footer element appears at the bottom of the page.
  • The href attribute inside a link element is relative to the current location of the file you are working in.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we awant around the content inside an element.
  • Inline CSS styles are applied only to a specific tag and its corrosponding element.
  • Internal CSS style sheets are CSS rules contained in the head of the HTML file that apply to that webpage.
  • External CSS style sheets are a separate CSS file that contains rules. You must link the CSS file to the HTML file for the CSS rules to apply to the webpage
  • All HTML files are boxes. The padding property changes the space around the content inside an element.
  • The margin property changes the space we want otside of an element.
  • A class attribute allows us to add styles to only certain elements that are assigned a class selector.
